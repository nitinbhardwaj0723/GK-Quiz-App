# 🧠 Quiz App (MCQ Based)

A modern and interactive Quiz Application built using **HTML, CSS, and JavaScript**.
This app presents multiple-choice questions (MCQs), tracks user score, and provides instant feedback with a clean UI.

---

## 🚀 Features

* 🎯 Multiple Choice Questions (MCQ)
* 📊 Real-time Score Tracking
* 📈 Progress Bar
* ✅ Correct / ❌ Wrong Answer Highlight
* 🔄 Restart Quiz Option
* ⚡ Direct Quiz Start (No Start Button)
* 🎨 Modern Glass UI Design

## 🚀 Future Improvements

* ⏱ Timer for each question
* 🔊 Sound effects
* 🏆 Leaderboard (LocalStorage / Firebase)
* 📱 Mobile Responsive UI
* 🎮 Difficulty Levels (Easy / Medium / Hard)
